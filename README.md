# A Hacker's AI Voice Assistant

Build your own **AI Voice Assistant from scratch using PyTorch**.

This project is part of a YouTube series that demonstrates how to build a complete AI voice assistant system step-by-step, including wake word detection, speech recognition, and other core components.

YouTube Series:
https://www.youtube.com/playlist?list=PL5rWfvZIL-NpFXM9nFr15RmEEh4F4ePZW
